# 让一人制公司真正拥有一套智能经营系统

一个面向 **一人制公司与 Agent 智能能力结合** 的开源创新共同体。

我们关心的，不是把 Agent 当成更多工具，
而是把它组织成一种真实可用的经营能力。

我们相信，未来重要的变化之一，不只是更多人开始使用 AI，
而是越来越多个人与小团队，将第一次真正拥有组织智能生产力与经营能力的方式。

这个项目围绕一条核心主线展开：

**主理人 → 主经营智能体（Chief Operating Agent）→ 七核岗位系统**

一句话概括：

**主理人决定方向，主经营智能体组织经营，七核岗位系统负责运行。**

---

## 我们在做什么

我们正在搭建一个开源共同体，探索：

- 一人制公司如何形成主权清晰的微型智能组织
- 主经营智能体如何成为智能世界中的经营代理中枢
- 七核岗位系统如何形成可判断、可协作、可执行、可审计、可沉淀的最小闭环
- 内容、研究、项目推进等场景如何率先跑通

---

## 仓库包含什么

- `manifesto/`：愿景、使命、Manifesto、社区总纲
- `foundations/`：方法骨架与智能经营系统总览
- `roles/`：七核角色系统与角色卡
- `workflows/`：七核协作工作流与交接清单
- `protocols/`：审计、留痕、复盘等协议
- `templates/`：任务进入与任务结构化模板
- `use-cases/`：内容、研究、项目推进样板
- `community/`：社区指南、讨论主题、共建路径
