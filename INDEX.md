# Start Here  
  
欢迎来到这个项目。  
  
如果你是第一次进入这里，  
最好的方式不是从头到尾顺读所有文件，  
而是按你的目标，从正确入口进入。  
  
这个仓库围绕一条非常明确的主线展开：  
  
**建立一个面向一人制公司与 Agent 智能能力结合的开源创新共同体，  
并逐步构建一套由主理人、主经营智能体与七核岗位系统组成的最小智能经营系统。**  
  
如果你只想先快速理解一句话，那就是：  
  
**我们不是在做一个人带七个 Agent 的系统，  
而是在构建一种由主理人掌握主权、由主经营智能体组织经营、由七核岗位系统完成运行的一人制公司智能经营系统。**  
  
---  
  
## 这个仓库里有什么  
  
这个仓库主要由八部分构成：  
  
### `manifesto/`  
项目的愿景、使命、Manifesto 与社区总纲。  
这里回答：**我们为什么做这件事。**  
  
### `foundations/`  
整个方法体系和智能经营系统的底层骨架。  
这里回答：**这套系统是什么结构。**  
  
### `roles/`  
七核岗位系统与角色卡。  
这里回答：**每个 Agent 负责什么。**  
  
### `workflows/`  
七核如何协作、交接和运行。  
这里回答：**这套系统怎么跑。**  
  
### `protocols/`  
监督审计、留痕、复盘等治理协议。  
这里回答：**这套系统如何可信、可控、可复盘。**  
  
### `templates/`  
任务进入、任务结构化、执行与复盘模板。  
这里回答：**真实任务如何进入系统。**  
  
### `use-cases/`  
场景样板。  
这里回答：**这套系统在真实任务里怎么用。**  
  
### `community/`  
社区指南、共建路径、讨论主题与社区价值。  
这里回答：**怎样参与、怎样共建。**
