# One-Person Company + Agent Seven-Core Model v1

## 一人制公司 + Agent 七核模型

如果主经营智能体是经营代理中枢，
那么七核岗位系统，就是这套智能经营系统真正的运行层。

七核模型的价值，不在于“把角色列出来”，
而在于：

**把一人制公司真正需要的最小经营岗位关系，组织成可运行、可交接、可闭环的系统。**

---

## 一、为什么是七核

我们不是为了复杂而复杂，
也不是为了看起来像“多 Agent 系统”而堆角色。

七核之所以成立，
是因为一人制公司在真实经营中，至少需要解决这七类问题：

1. 这件事值不值得做  
2. 这件事怎么拆、怎么排、怎么组织  
3. 这件事需要哪些信息和研究支撑  
4. 这件事如何变成可读、可交付、可表达的成果  
5. 这件事如何真正执行到现实中  
6. 这件事如何被监督、纠偏、审计  
7. 这件事如何留下记忆与复利资产  

这七类问题，正好对应七核。

---

## 二、七核分别是什么

### 1）判断 Agent
负责：
- 判断问题是否成立
- 判断现在该不该做
- 判断优先级与风险
- 识别不可逆路径与高成本路径

它的核心价值是：

**减少错误动作。**

---

### 2）编排 Agent
负责：
- 把任务拆成阶段与节点
- 设计交接关系
- 安排顺序与节奏
- 明确主理人仲裁点

它的核心价值是：

**让任务不靠临场 improvisation，而靠结构推进。**

---

### 3）研究 Agent
负责：
- 搜集资料
- 提炼关键事实
- 对比来源
- 输出结构化研究输入
- 标注不确定性

它的核心价值是：

**让系统更接近真实，而不是更接近信息噪音。**

---

### 4）表达 Agent
负责：
- 形成文稿
- 整理说明
- 输出方案
- 压缩复杂内容
- 做受众适配

它的核心价值是：

**把原材料变成可使用的成果。**

---

### 5）执行 Agent
负责：
- 推进动作
- 调用工具
- 执行外部动作
- 更新状态
- 回传结果

它的核心价值是：

**让结果真正发生在现实中。**

---

### 6）监督审计 Agent
负责：
- 检查越界与漏项
- 检查责任链
- 检查关键节点是否留痕
- 识别质量偏移
- 输出治理反馈

它的核心价值是：

**让系统在变快之后，依然值得信任。**

---

### 7）记忆 Agent
负责：
- 归档过程
- 沉淀模板
- 更新规则
- 保存高价值来源
- 提炼可复用经验
- 形成经营资产

它的核心价值是：

**让系统不是一次次重来，而是一次次变强。**

---

## 三、七核不是七个孤立角色

七核最重要的，不是“七个名字”，
而是它们之间的运行关系。

一个基本闭环是：

**判断 → 编排 → 研究 → 表达 → 执行 → 监督审计 → 记忆**

这里面每一核都不是单独存在的。

### 判断约束编排
如果判断没做好，后面的编排就容易跑偏。

### 编排组织研究、表达与执行
如果没有编排，研究会散，表达会乱，执行会失序。

### 监督审计约束执行
如果执行没有被监督，系统会越来越失真。

### 记忆承接整个链路
如果没有记忆，整个系统没有复利。

---

## 四、七核模型与主理人的关系

七核不是主理人的替代层，
而是主理人的智能岗位系统。

主理人仍然负责：
- 目标
- 边界
- 最终优先级
- 最终仲裁
- 最终责任

七核负责的是：
- 把任务组织起来
- 把运行链路打通
- 把结果沉淀下来

所以准确的说法不是：

“七个 Agent 替主理人做事”

而是：

**七核岗位系统，帮助主理人拥有一个最小智能经营结构。**

---

## 五、七核模型与主经营智能体的关系

主经营智能体不替代七核，
也不和七核并列。

它的位置在七核之上。

关系是：

**主理人  
→ 主经营智能体  
→ 七核岗位系统**

主经营智能体负责：
- 调度七核
- 管理顺序
- 汇总状态
- 风险上浮
- 经营反馈

七核负责：
- 真正把系统跑起来

所以：

**主经营智能体是经营中枢，  
七核岗位系统是运行层。**

---

## 六、为什么七核适合一人制公司

因为一人制公司最大的难点，不是有没有能力，
而是能力能不能被组织起来。

七核模型特别适合一人制公司的原因在于：

### 1）它不依赖复杂层级
七核靠结构，不靠层级。

### 2）它能承接主权集中
主理人仍是中心，但不用亲自管理全部细节。

### 3）它可以最小化运行
七核可以先从轻量模式开始，不必一次全开。

### 4）它天然适合模板化与复利
七核每一核都能沉淀成模板、规则、卡片与工作流。

---

## 七、轻量理解

如果用更大白话的方式说，七核就是：

- 有人先判断
- 有人负责拆解和安排
- 有人去查和补信息
- 有人把东西写清楚
- 有人把事真正推进
- 有人盯着别跑偏
- 有人把经验记下来

这七件事，传统上往往都压在一个人脑子里。
七核模型做的事，就是把它们显性化、结构化、系统化。

---

## 八、一句话总结

**七核模型，是一人制公司智能经营系统的最小岗位运行层。**
