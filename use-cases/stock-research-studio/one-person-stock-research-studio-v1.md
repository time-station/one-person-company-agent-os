# One-Person Stock Research Studio v1

## 一人股票研究工作室样板

股票研究是所有研究场景中对**判断质量**要求最高的场景之一。

它不只是搜集信息，不只是整理财报，
它最终要回答一个问题：

**这家公司，当前的价格，值不值得持有或买入？**

这是一个需要：

- 基本面分析
- 估值框架
- 行业理解
- 管理层判断
- 财务核查
- 市场情绪识别
- 风险定价

七种能力同时在线的判断行为。

因此，一人股票研究工作室，是七核系统最适合发挥深度价值的场景之一。

---

## 一、这个样板在解决什么问题

个人投资者与专业研究者都会遇到这些问题：

- 看了很多财报，但没有形成真正判断
- 研究结论容易被市场情绪带偏
- 逻辑说服了自己，但没有做压力测试
- 买入逻辑不清晰，持有过程中容易被噪音动摇
- 每次研究都从零开始，没有形成方法资产
- 研究结论和风险边界之间缺乏清晰断点

所以，一人股票研究工作室的目标不是"看更多股票"，
而是：

**让股票研究工作开始具备最小专业经营系统。**

---

## 二、样板的基本结构

一人股票研究工作室的最小结构是：

**主理人（投资者）
→ 主经营智能体
→ 七核岗位系统
→ 股票研究任务闭环**

其中：

### 主理人（投资者）负责
- 决定研究的公司或行业方向
- 设定个人投资框架与原则
- 决定仓位逻辑与风险容忍边界
- 对最终投资决策负责
- 定义什么样的结论叫"够用"

### 主经营智能体负责
- 把研究意图翻译成任务清单
- 决定研究深度（轻看 / 标准研究 / 深研究）
- 调度七核岗位
- 汇总研究进度与状态
- 暴露研究缺口与逻辑断点
- 输出阶段性研究反馈
- 提醒主理人不要跳过关键环节

### 七核岗位系统负责
- 判断这家公司是否值得进入研究
- 拆解研究任务结构
- 搜集并筛选财务与行业数据
- 形成结构化研究报告
- 推进决策与跟踪动作
- 审计逻辑是否存在失真
- 沉淀研究模板与投资知识库

---

## 三、适合什么样的股票研究场景

这个样板最适合：

- 个人价值投资者（需要构建完整研究体系）
- 专业基金分析师（需要提高研究效率与质量）
- 创业者研究竞争对手上市公司
- 企业战略部门研究同行公司
- 研究员训练期快速建立研究能力
- 任何想把股票研究做成方法资产、而非一次性工作的人

它特别适合那些：

**不只是想"看个股"，而是想构建一套长期有效的股票研究系统的人。**

## 四、股票研究的核心分析框架

在进入工作链路之前，需要理解股票研究的五大核心分析维度：

### 维度一：商业模式理解
- 公司怎么赚钱？收入来源与利润结构是什么？
- 商业模式是否具备可持续性？
- 定价权从哪里来？护城河是否真实存在？
- 这个商业模式在未来5年是变好还是变坏？

### 维度二：财务质量分析
- 营收增长是否可持续？增速趋势如何变化？
- 利润率水平与同行相比处于什么位置？
- 现金流与净利润是否匹配？自由现金流如何？
- 负债结构是否健康？资产质量有无隐患？
- 资本配置历史是否显示出管理层的能力与诚信？

### 维度三：行业竞争格局
- 这家公司在行业里处于什么位置？
- 行业是否进入存量竞争？增量空间还有多大？
- 核心竞争对手是谁？竞争优势如何比较？
- 行业的进入壁垒、转换成本、网络效应是否支撑估值？
- 行业监管风险、政策变量是否需要纳入定价？

### 维度四：管理层与治理
- 创始人还是职业经理人？激励机制是否对齐长期价值？
- 过去的资本配置决策是否有良好的记录？
- 管理层的沟通是否诚实、具体，还是充满营销话语？
- 股权结构是否合理？大股东行为是否值得信任？
- 有无重大诉讼、关联交易或审计异常需要关注？

### 维度五：估值与安全边际
- 当前用哪种估值方法最合适（PE / EV/EBITDA / DCF / P/B）？
- 基于合理假设的内在价值是多少？
- 当前市场价格对应的隐含增速假设是否合理？
- 安全边际是否足够？下行空间与上行空间之比是多少？
- 市场共识在哪里？认知差在哪里？

## 五、最小工作链路

一轮标准股票研究任务，最小可以这样跑：

---

### 第一步：研究进入与标的筛选

主理人提出研究标的，或设定筛选条件，例如：

- 直接指定：「我想研究 $NVDA / 宁德时代 / 腾讯」
- 条件筛选：「帮我找出过去5年 ROE > 20% 且负债率 < 40% 的消费品公司」
- 行业路径：「我想了解中国 AI 算力行业，从哪几家公司入手最好？」

**主经营智能体** 负责：
- 评估标的研究价值（行业地位、流动性、信息可获取性）
- 分级：初步筛查 / 标准研究 / 深度研究
- 确认研究目的：长期持有研究 vs. 交易机会研究 vs. 行业地图研究

---

### 第二步：判断前置 — 值不值得研究？

**判断 Agent** 先做第一轮过滤，回答：

- 这家公司是否在主理人的能力圈范围内？
- 是否存在明显的结构性问题导致不值得深入（如：账目不透明、流动性太低）？
- 当前是否是研究这家公司的好时机？（重大事件窗口 / 财报季 / 估值极端期）
- 研究的核心问题是什么？（收益能力验证 / 估值重估 / 催化剂识别）

如果判断不值得研究，直接止损，避免沉没成本陷阱。
如果值得研究，进入结构化分析。

---

### 第三步：研究结构化 — 拆出核心问题

**编排 Agent** 把研究任务拆成可执行的结构，例如：

```
公司：[公司名称]
研究目的：[长期持有判断 / 短期催化剂识别 / 行业对比]
核心问题：
  1. 商业模式变化了吗？护城河是否在强化？
  2. 最新财报的收入质量与现金流如何？
  3. 当前估值对应的增长预期是否合理？
  4. 主要风险是什么？风险对股价的影响量级如何？

研究边界：
  - 本轮重点研究范围（如：只研究国内业务，暂不分析海外）
  - 不需要覆盖的方面（节约时间）

证据需求：
  - 财务数据：近5年财报 + 最新季报
  - 行业数据：市占率、竞争对手财务对比
  - 管理层信息：最近3次财报电话会议摘要
  - 估值参考：历史PE区间、同行估值倍数
```

---

### 第四步：资料与数据搜集

**研究 Agent** 负责搜集和整理：

**财务数据层**
- 过去5-10年的营收、净利润、毛利率、净利率趋势
- 自由现金流计算与质量分析（FCF = 经营活动现金流 - 资本支出）
- 资产负债表健康度：现金、负债、商誉、应收账款质量
- ROIC（投入资本回报率）趋势：持续 > 15% 是质量标志
- 分红历史与股票回购记录

**行业竞争数据层**
- 市场份额变化（近3年）
- 核心竞争对手对标分析（至少3家）
- 行业增速 vs. 公司增速对比
- 行业进入壁垒定性评估

**管理层信息层**
- 最近2-3次财报电话会议（Earnings Call）核心表述
- 大股东持股变化（近6个月）
- CEO / CFO 背景与过往决策记录
- 近期重大战略动作（并购、分拆、扩张）

**估值数据层**
- 历史PE / PB / EV/EBITDA 区间（近5年）
- 当前估值在历史区间中的百分位
- 分析师预期与共识目标价
- 期权市场隐含波动率（反映市场定价的不确定性）

---

---

### 第六步：执行与决策

**执行 Agent** 负责把研究推进到现实动作，例如：

- 纳入持仓组合评估
- 设定跟踪触发条件（价格、财报日、重大事件）
- 进入观察清单或直接买入
- 给出具体的仓位建议（初始仓位 / 加仓条件）
- 同步到投资组合跟踪系统

---

### 第七步：监督审计

**监督审计 Agent** 做研究质量审查：

**逻辑一致性检查**
- 结论是否和证据对得上？
- 有没有"先有结论，再找证据"的确认偏误？
- 关键假设是否经过压力测试？

**常见认知偏差检查**
- 是否过于依赖近期信息（近因效应）？
- 是否对熟悉的公司过于宽容（熟悉偏误）？
- 是否把"好公司"等于"好股票"（质量陷阱）？
- 估值分析是否锚定在买入价上（锚定效应）？

**风险覆盖检查**
- 尾部风险是否被识别（黑天鹅场景）？
- 行业系统性风险是否被纳入？
- 杠杆风险、流动性风险是否被评估？

**信息质量检查**
- 数据来源是否可信？
- 财报是否存在审计异常需要关注？
- 分析师预期是否存在明显的系统性偏差？

---

### 第八步：记忆与沉淀

**记忆 Agent** 负责把本轮研究沉淀成资产：

- **公司档案**：基本数据 + 商业模式摘要 + 历史研究记录
- **行业地图**：竞争格局 + 关键玩家 + 变量跟踪清单
- **估值框架**：哪种估值方法最适合该公司的历史记录
- **判断模式**：哪类公司研究结果最好、哪类最容易犯错
- **来源体系**：哪些信息来源最高效、哪些是噪音
- **失误记录**：哪次研究判断错了，原因在哪里

## 六、不同研究深度的标准

### 初步筛查（1-2小时）
适用于：快速决定是否值得深入研究

必须覆盖：
- 商业模式一句话理解
- 营收/利润近3年趋势
- 当前估值水平（贵/合理/便宜的直觉感知）
- 是否在能力圈内

输出：**进入 / 暂不进入 / 列入观察** 三选一

---

### 标准研究（3-6小时）
适用于：初步看好后的完整评估

必须覆盖：
- 五大分析维度全部触及
- 财务核心指标完整分析
- 至少2家核心竞争对手对比
- 初步估值区间测算
- 核心风险清单

输出：**结构化研究报告 + 投资决策建议**

---

### 深度研究（1-3天）
适用于：重仓持有前的深度调研，或行业全面梳理

必须覆盖：
- 10年以上财务趋势分析
- 完整的 DCF 估值模型
- 管理层全面调研（历史发言记录分析）
- 行业竞争格局深度研究
- 多情景假设与压力测试
- 全面风险矩阵

输出：**完整深度研究报告 + 投资组合配置建议**

## 七、主经营智能体在股票研究中的特殊作用

股票研究最容易出现两种经营失控：

### 1）研究过度而不决策
收集了大量信息，但始终无法形成判断，陷入"分析瘫痪"。

**主经营智能体的应对**：
- 明确研究目的（是要做决策，不是要做完美报告）
- 设定研究截止时间
- 强制生成初步结论，哪怕是"暂时不够信心，列入观察"

### 2）决策过快而研究不足
被市场情绪感染，跳过必要分析步骤直接买入。

**主经营智能体的应对**：
- 检查七核是否全部过了一遍
- 确认判断 Agent 的初步过滤结论
- 确认审计 Agent 的偏差检查已完成

---

## 八、股票研究常见的认知陷阱

做股票研究最容易踩的几个坑：

### 陷阱一：好公司 = 好投资
一家公司再好，如果估值太贵，也不是好投资。
**判断标准**：买入时的价格和内在价值的关系，比公司本身的质量更重要。

### 陷阱二：近期财报代表全部
股市是前瞻性的。当财报很好时，往往已经反映在价格里了。
**判断标准**：比的不是"好不好"，而是"是否比市场预期好"（预期差）。

### 陷阱三：叙事替代财务分析
宏大的产业叙事经常掩盖差劲的财务现实。
**判断标准**：任何叙事，都需要财务数字来支撑，没有数字的叙事是情绪，不是研究。

### 陷阱四：忽略资本配置历史
管理层的过去决策，比他们的现在表态更可信。
**判断标准**：看过去5年的并购、分红、回购、投资决策质量。

### 陷阱五：止损条件不清晰
买入时没有设定"如果哪些假设被证伪就离场"的条件。
**判断标准**：每笔买入都需要有明确的持有逻辑和失效条件。

## 九、最容易沉淀的研究资产

一人股票研究工作室特别容易沉淀下面这些高价值资产：

### 1）公司知识库
研究过的公司档案，包括：商业模式摘要、历史估值区间、关键变量。

### 2）行业地图
不同行业的竞争格局、核心玩家、分析框架。

### 3）估值经验库
哪类公司用哪种估值方法最有效，历史估值中枢数据。

### 4）失误记录本
哪次判断错了、为什么错、下次如何避免。这是最贵的资产。

### 5）高质量信息来源清单
哪些分析师、研报、媒体、数据库最值得长期跟踪。

### 6）个人投资框架文档
自己的能力圈边界、选股标准、仓位管理原则。

## 十、最小使用方式

如果今天就想跑一轮，不需要把全部系统都搭好。

最小可以这样：

1. 主理人提出一个想研究的公司或行业方向
2. 用判断 Agent 的逻辑做初步过滤（值不值得研究？）
3. 用编排 Agent 拆出核心问题（这次研究最想回答什么？）
4. 做一轮聚焦数据搜集（财务3-5年趋势 + 估值水平 + 核心竞争格局）
5. 用报告模板形成一版研究摘要
6. 用审计 Agent 的偏差清单做一遍自检
7. 输出结论：买入 / 持有 / 观察 / 回避 + 核心理由
8. 用复盘卡记录：这轮研究最值得沉淀的是什么

只要这 8 步跑通，一人股票研究工作室就已经开始活起来了。

## 十一、一句话总结

**一人股票研究工作室样板，是把股票研究从"看财报 + 直觉判断"升级为"由投资者掌握判断标准、由主经营智能体组织研究节奏、由七核岗位系统完成分析闭环"的最小专业投资研究系统。**

它不是为了替代你的判断，

**更结构、更可信、更可复盘、更能积累。**
